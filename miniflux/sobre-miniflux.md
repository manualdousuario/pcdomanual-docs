# Sobre o Miniflux

O Miniflux é um agregador de feeds RSS minimalista e focado na leitura. Projetado para ser leve e rápido, ele remove todas as distrações, entregando o conteúdo de seus sites, blogs e canais favoritos em uma interface limpa e cronológica.

Ao contrário de redes sociais ou agregadores comerciais, o RSS não utiliza algoritmos para decidir o que você deve ler. Você tem total controle sobre seu fluxo de informações.

## O que é RSS?

RSS (_Really Simple Syndication_) é um protocolo aberto que permite acompanhar atualizações de sites sem precisar visitá-los individualmente. É a maneira mais eficiente de consumir conteúdo da web de forma decentralizada.

> No **Manual do Usuário** temos um artigo completo [explicando como usar feeds RSS](https://manualdousuario.net/como-usar-feeds-rss/).

## Principais recursos

O Miniflux foi concebido com um conjunto de funcionalidades que priorizam a eficiência técnica e a experiência de leitura sem distrações.

### Minimalismo

A interface do Miniflux é intencionalmente simples. O objetivo é a leitura. Não há anúncios, pop-ups, rastreadores ou elementos visuais desnecessários. O layout é otimizado para legibilidade em qualquer dispositivo.

### Privacidade

O Miniflux respeita sua privacidade. Ele não rastreia seu comportamento, não vende seus dados e não analisa seus hábitos de leitura para fins publicitários.

### Desempenho

Construído em linguagem Go, o Miniflux é extremamente eficiente. Ele carrega instantaneamente e consome poucos recursos, garantindo uma experiência fluida mesmo com centenas de assinaturas.

### Integração

O Miniflux oferece uma API robusta e é compatível com a API Fever. Isso significa que você pode usá-lo como "backend" e consumir seus feeds através de diversos aplicativos de terceiros populares.
